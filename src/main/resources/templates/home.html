칼칼 페이지는? Calorie Calculate

내가 먹은 음식에 대한 영양소를 계산할 수 있는 웹 서비스 입니다.

식약처 정보를 기반으로 만들어졌습니다.

현재는 beta 버전으로 탄단지 등의 기본 정보만을 제공하며, 추후 비타민, 무기질 등이 추가될 예정입니다.

사용방법

  1. 음식 검색
    • 원하는 음식을 검색
    • 기본검색 / 많은 검색 / 더 많은 검색
  2. 내 음식
    • 1단계 검색결과에서 음식명을 클릭하여 '내 음식'으로 저장
    • 또는 '내 음식' 페이지에서 '생성'을 클릭하여 직접 입력하여 사용
  3. 내 식단
    • 내 음식을 모아서 내 식단을 구성
    • 먼저 내 식단을 '생성' 한 후, 내 음식을 체크하여 식단에 '추가'
    • 식단의 영양소 합계를 계산

(내 음식과 내 식단을 사용하기 위해서 로그인이 필요합니다.)

(beta 버전에서 id, password 만으로 간단히 회원가입 가능!)
